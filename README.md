# CLASSES

## Container
#### Methods:
    AppendChild - Add a child to this container
    RemoveChild - Remove child from container
    GetChildren - Get a table of children
#### Events:
    ChildAdded - Fires when a child is added
    ChildRemoved - Fires when a child is removed
    LineageChanged - Fires when either a child is added or removed
#### Properties
    Enabled - Determines whether the container is used or not
    Name - Name of container
    Children - Table containing the children of the container
